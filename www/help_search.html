The same strategies used in PubMed also can be used in this app, like Boolean operators, search tags, filter strategies, etc. This means that using your search query you could filter you search in different ways.
E.g. to create a search query for articles about COVID-19 and anemia published during the last year you could create a string like: 
(anemia OR anaemia) AND "last year"[dp] AND covid
Or if you want to search review articles about diabetes in Spanish published between 2010 and 2015 you could create a search query like 
diabetes AND spanish[la] AND 2010:2015[dp] AND review[pt]
More information about how to search can be found in the PubMed User Guide.
